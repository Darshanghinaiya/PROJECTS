Student Attendance Tracker

Tech Stack: HTML, Tailwind CSS, JavaScript, File System Access API

Developed a browser-based attendance management system to handle courses, student enrollment, and attendance tracking without backend support.
Implemented dynamic course and student management with validation to prevent duplicate entries.
Designed an interactive attendance marking system with date and course selection, ensuring no duplicate records per session.
Built a reporting module to calculate attendance statistics (present, absent, percentage) with conditional highlighting for low attendance.
Integrated local file handling using JSON for persistent data storage (load/save functionality).
Created a responsive, dark-themed UI using Tailwind CSS with tab-based navigation for improved user experience.
Utilized efficient state management and real-time DOM updates for seamless performance.
