# 🔳 QR Code Generator with Logo (Python + Tkinter)

A simple and user-friendly **QR Code Generator** built using Python.  
This application allows users to generate QR codes from text or URLs and embed a **custom logo at the center**.

---

## 🚀 Features

- Generate QR codes from text or URLs  
- Embed logo inside QR code (centered)  
- Live preview inside the application  
- Save QR code as PNG file  
- Error handling for empty input  
- Clear input and preview easily  
- High error correction ensures QR remains scannable  

---

## 🛠️ Technologies Used

- Python  
- Tkinter (built-in GUI library)  
- qrcode library  
- Pillow (PIL) for image processing  

---

## 📦 Prerequisites

Make sure you have:

- Python 3.x installed  

Install required library:

pip install qrcode[pil]

---

## 📁 Project Setup

Keep all files in the same folder:

QR-Code-Generator/
- main.py  
- logo.png  
- README.md  

---

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/Darshanghinaiya/PROJECTS.git

2. Navigate to the project folder:

cd PROJECTS/Python/QR_Code_generator

3. Run the program:

python main.py

---

## ⚠️ Important Notes

- Place your logo file in the same directory  
- Logo file name must be: `logo.png`  
- Use a simple and transparent PNG logo for best results  
- QR uses high error correction (H) so it works even with logo  

---

## 💡 How It Works

- Takes input (text or URL) from user  
- Generates QR using qrcode library  
- Applies high error correction  
- Inserts logo at center  
- Displays preview  
- Allows saving as PNG
