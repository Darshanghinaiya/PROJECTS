## 🔳 QR Code Generator with Logo

A simple GUI-based application built using Python that generates QR codes from text or URLs and embeds a custom logo at the center.
### 🚀 Features
- Generate QR codes instantly  
- Embed logo inside QR  
- Live preview  
- Save as PNG

### 🛠️ Tech Stack
- Python  
- Tkinter  
- qrcode  
- Pillow
